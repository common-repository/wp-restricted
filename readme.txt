wp admin ban for all ip except one ip .
 
== Description ==
This plugin is restricted all ip except one ip . if you want open wp admin with particular IP.

== Installation ==

1. Open `wp-content/plugins` Folder
2. Put: `Folder: wp-restricted`
3. Activate `WP-Restricted` Plugin
4. Go to `WP-Admin -> Settings -> Wp Restricted` to configure the plugin

== Installation ==

The plugin can be use just by installing it by wordpress admin or upload manually.
